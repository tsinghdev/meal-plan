# Meal Plan

A weekly meal planning page designed to help track a balanced diet schedule.

## About

This is a personal hobby project that serves as a lightweight record keeper, optimized for use as a home screen web app on iPhone. It requires no installation and runs entirely in the browser.

**Live URL:** https://tsinghdev.github.io/meal-plan/meal-plan.html
